# Yoink

Native-feel cross-platform FTP/SFTP client. Built with Tauri v2 + Rust + React 19.

Grab files fast. Native on macOS, Windows, Linux. Credentials in the OS keychain. No server, no account, no cloud.

## Status

Alpha — scaffold in place.

## Dev

```bash
bun install
bun run tauri dev
```
